ERP Connect for a US based eCommerce leader


About The Project :


Client is a leading company based out if the USA, selling Beauty and Skin Care products. They have many sales channels and one of them is their e-commerce website. Order Fulfillment for all sales channels is done in a single warehouse..


Scope Of The Project :


  • Sync order to ERP and 3PL systems
  • Sync shipment data from 3PL to eCommerce
  • Sync Inventory data from ERP to e-commerce system
  • Dashboard to check the status of Orders and various sync processes

Challenges :


  • Handle one transaction at a time to ERP

    • Built a custom lock mechanism to handle all the transactions to be sent to ERP system in a sequential manner


    Used re-try mechanism to reinitiate the failed transaction



  • Handle API throttling limit

    • The e-commerce REST APIs had a throttling limit and there was a possible case where it got exceeded since the same credentials were used by many applications


    Addition of custom logic to handle the limit and number of API calls to be made in an hour to avoid exhausting the limit
