Sales Cloud for Barbecue Society


About The Project :


The client is an organization based out of the USA and was founded to promote traditions, lifestyle and skills associated with barbecuing. They offer classes related to barbecuing and outdoor cooking and conduct contests and social gatherings for promotions.


Scope Of The Project :


  • Integration with external system to pull Contest and Results data
  • Import functionality for result declaration
  • Rank calculation logic and trigger mechanism for each contest
  • Community page for result declaration

Challenges :


  • Unavailability of Upsert or Update of data using multiple Unique key fields in Data loader

    • Built a Custom Import functionality to Import a CSV to Insert, Update or Upsert of data with multiple keys fields as matching criteria



    Solution Implemented :


  • Custom Import functionality was built to import Contest, Team and Contest Score data via CSV file
  • A scheduled batch process is built to pull Contest and Contest Results data from KCBS and loaded into Salesforce Custom Objects
  • A function will be built to calculate NEBS Score of each team for all Contests and respective triggers were added to initiate it
  • A function will be built to re-calculate Rank based on NEBS score for all Year run Races and respective triggers were added to initiate it
  • Community pages will be built using Lightning to display the Contest Scores and Race Results

    • Technology Stack :


    Salesforce Apex classes, Flow builder, Process builders
    API Integration
